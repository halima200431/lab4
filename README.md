# 📱 Présentation de l'Application
Cette interface Android repose sur une architecture moderne à fragments interchangeables, habillée d'un design élégant aux tons Beige et Marron. Elle se divise en deux modules interactifs :

## Fragment 1 : Interaction Statique
Ce module accueille l'utilisateur avec un affichage textuel clair et un bouton d'action principal, idéal pour la validation ou l'envoi de commandes.

## Fragment 2 : Contrôle Dynamique
Ce module intègre une SeekBar (barre de sélection) permettant de modifier des paramètres en temps réel. La valeur affichée s'ajuste instantanément selon la position du curseur.

# 🎨 Identité Visuelle
L'application utilise une palette "Nature" pour un confort visuel optimal :

Fonds : Beige sable pour la clarté.

Éléments : Marron chocolat pour un contraste doux et professionnel.
